# Financial KPI Dashboard (Dash)

Mini-Dashboard zur Visualisierung von Finanz-KPIs mit **Python, Pandas, Plotly, Dash**.

## Features
- Upload oder Nutzung der Demo-Daten (`data_demo.csv`)
- KPIs: Umsatz, Kosten, Profit, Profit-Marge
- Liniendiagramm (Zeitreihe), Balkendiagramm (Vergleich), Filter (Jahre)

## Stack
Python 3.10+, Pandas, Plotly, Dash

## Schnellstart
```bash
pip install -r requirements.txt
python app.py
